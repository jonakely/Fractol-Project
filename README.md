Fractol-Project
Projet réalisé à l’École 42 permettant de générer différents types de fractales graphiques en utilisant MiniLibX, avec un focus sur la gestion des ressources et la programmation en C.

Ce projet illustre :
La programmation bas niveau en C, avec contrôle de la mémoire et optimisation des performances.
L’implémentation d’algorithmes de génération de fractales (Mandelbrot, Julia, etc.).
La gestion graphique et l’affichage interactif grâce à MiniLibX.
Le respect des bonnes pratiques de développement et de structuration du code.

Installation et Compilation :
1 - Cloner le dépôt :
git clone https://github.com/jonakely/Fractol-Project.git
cd fractol

2 - Compiler le projet avec Make :
make

3 - Lancer le programme :
./fractol <type_de_fractale>
(Exemples de types de fractales : mandelbrot, julia ou (julia [option 1] [option 2]))
