# Fractol-Project
Projet permettant de générer différents types de fractales avec MiniLibX, en mettant l’accent sur la bonne gestion des ressources et l’utilisation du langage C.

Ce projet implémente divers algorithmes de génération de fractales. Il illustre la gestion fine des ressources et la programmation graphique en C, un langage bas niveau privilégiant la performance et le contrôle mémoire.
